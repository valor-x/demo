## 提交友链申请

必须为使用本主题的用户，且博客内容健康，无违法违规内容。

按照自己的博客类型，然后在分类的底部进行添加。

fork [本仓库](https://github.com/wleelw/solitude.wzsco.top)，修改 `source/links/index.md` 文件，按照如下格式添加你的链接：

```yaml
- name:   【博客名称】
  link:   【博客地址】
  tag:    【标签】
  descr:  【博客简介】
  color:  【vip、speed】
  avatar: 【博客头像】
  topimg: 【博客预览图】
```
